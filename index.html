🔍 環境診斷
點擊「執行」自動偵測 Chrome 除錯環境。
啟動 Chrome 除錯模式
Windows
chrome.exe --remote-debugging-port=9222

macOS
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222

Linux
google-chrome --remote-debugging-port=9222
⬡ Extension 安裝指引
安裝 Extension 後，本工具可在 EXT 模式下直接控制 Chrome，無需開啟除錯埠。
點擊下方「⬇ 下載 Extension」按鈕，下載 mcp-rpa-extension.zip
解壓縮到任意資料夾（例如桌面）
在 Chrome 網址列輸入 chrome://extensions/
開啟右上角「開發人員模式」
點擊「載入未封裝項目」→ 選擇解壓縮後的資料夾
工具列出現 ⬡ 六角形圖示即安裝成功
回到本頁面，底部切換到「⬡ EXT」模式，點「◎ 偵測 Extension」確認連線
